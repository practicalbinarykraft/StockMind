# Мок данные для демонстрации

Этот модуль содержит мок данные для демонстрации работы системы разработчикам.

## Файлы

- `mock-scripts.ts` - мок данные для сценариев с примерами итераций AI редактирования
- `mock-conveyor.ts` - мок данные для конвейера с прогрессом генерации

## Использование

### Мок скрипты

```bash
# Получить все мок скрипты
GET /api/scripts/mock

# Получить мок скрипты по статусу
GET /api/scripts/mock?status=completed

# Получить один мок скрипт
GET /api/scripts/mock/mock-script-1
```

### Мок данные конвейера

```bash
# Получить все мок данные конвейера
GET /api/conveyor/mock

# Получить мок данные по статусу
GET /api/conveyor/mock?status=in_progress

# Получить один мок элемент
GET /api/conveyor/mock/conveyor-1
```

## Примеры данных

### Скрипт с итерациями

Мок скрипты показывают процесс улучшения сценариев через итерации:

1. **Первая итерация** - начальный вариант сценария с оценкой редактора
2. **Вторая итерация** - улучшенный вариант с учетом замечаний, показывает `changes` для каждой сцены
3. **Третья итерация** (если есть) - финальный вариант

### Конвейер

Мок данные конвейера показывают:
- Текущий этап генерации (`scriptwriter`, `editor`, `completed`)
- Прогресс в процентах
- Статистику по итерациям (средний и лучший скор)

## Структура данных

Все мок данные соответствуют типам фронтенда и могут использоваться напрямую без трансформации.
